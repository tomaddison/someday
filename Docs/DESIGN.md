# Design

The feel of the app is part of the product. A few principles guide every screen:

- **Calm over urgent** - no badges, no streaks, no guilt. The app never shouts.
- **Depth over breadth** - a small number of features done beautifully.
- **Ritual over routine** - interactions should feel meaningful, not mechanical.
- **Animations are first-class** - native spring physics throughout; every transition is intentional.
- **The solar system is alive** - the home screen breathes, orbits, and drifts like a living map of the user's aspirations.

**Typography:** EB Garamond SemiBold for display text and headlines; HarmonyOS Sans for body and supporting UI.
